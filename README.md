# Infracredit Conflict Declaration

A web application for managing conflict of interest declarations.

## Prerequisites

- Node.js (LTS version)
- npm or pnpm

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

```bash
npm run build
```

## Testing

```bash
npm run test:e2e
```
